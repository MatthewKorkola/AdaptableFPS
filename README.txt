- Shift (hold): run
- Space: jump/double jump
- Left Click: shoot
- Right Click: zoom in/out
- WASD: player movement
- Move Mouse: camera movement
- Tab: switch weapons
- Escape: pause/unpause
- L: Reset level to default spawn point

Note that server.py has to be running for the model to work properly.
Note that the pathways in test_model.py need to be modified to match where you have placed them.
